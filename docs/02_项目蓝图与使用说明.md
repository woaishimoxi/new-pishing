## 项目蓝图与使用说明

---

## 一、项目概述

### 1.1 项目背景

钓鱼邮件攻击频次年增超30%，中小企业因安全投入少、IT能力不足成为主要目标。现有解决方案存在以下问题：

| 方案类型 | 问题 |
|----------|------|
| 商业产品 | 年费高昂（数万元），依赖高配硬件 |
| 开源工具 | 需手动配置，缺乏溯源能力 |
| 深度学习 | 资源消耗大，不适合低配环境 |

### 1.2 项目目标

构建一个**检测 + 溯源 + 可视化**三位一体的安全防护系统：

```
┌─────────────────────────────────────────────────────────┐
│                    系统目标                              │
├─────────────────────────────────────────────────────────┤
│  ✓ 高准确率：≥90%                                       │
│  ✓ 低误报率：≤5%                                        │
│  ✓ 低资源：2核4G即可运行                                 │
│  ✓ 一键部署：3步操作，10分钟完成                          │
│  ✓ 智能配置：自动推荐最优参数                            │
└─────────────────────────────────────────────────────────┘
```

## 四、使用说明

### 4.1 环境要求

| 项目 | 最低要求 | 推荐配置 |
|------|----------|----------|
| CPU | 2核 | 4核 |
| 内存 | 4GB | 8GB |
| 硬盘 | 10GB | 20GB |
| Python | 3.8+ | 3.9+ |
| 操作系统 | Windows/Linux/Mac | Linux |

### 4.2 安装部署

#### 方式一：一键部署（推荐）

**Windows系统**：
```cmd
cd scripts\startup
deploy.bat
```

**Linux/Mac系统**：
```bash
cd scripts/startup
chmod +x deploy.sh
./deploy.sh
```

#### 方式二：Docker部署

```bash
cd scripts/startup
docker-compose up -d
```

#### 方式三：手动部署

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 创建目录
mkdir -p data/uploads data/logs logs uploads

# 3. 启动系统
cd backend
python run.py
```

### 4.3 访问系统

部署成功后，打开浏览器访问：

```
http://localhost:5000
```

### 4.4 功能使用

#### 4.4.1 邮件检测

1. **手动输入**：在界面中粘贴邮件内容
2. **上传文件**：上传.eml或.msg格式的邮件文件
3. **自动获取**：配置邮箱后自动收取检测

#### 4.4.2 查看检测结果

- 主界面：查看统计概览和邮件列表
- 点击邮件：查看详细检测报告
- 报告内容：风险等级、置信度、模块评分、溯源信息

#### 4.4.3 系统配置

点击右上角齿轮图标，配置：
- VirusTotal API Key（可选）
- 邮箱服务器信息（可选）

#### 4.4.4 智能调优

点击"系统配置" → "智能调优"，系统将自动分析邮件模式并推荐最优参数。

## 八、常见问题

### Q1: 如何获取VirusTotal API Key？

1. 访问 https://www.virustotal.com
2. 注册账号
3. 在设置中获取API Key
4. 填入系统配置

### Q2: 邮箱配置失败怎么办？

1. 确认邮箱已开启IMAP/POP3服务
2. 使用授权码而非登录密码
3. 检查端口是否正确（IMAP:993, POP3:995）

### Q3: 如何提高检测准确率？

1. 配置VirusTotal API Key
2. 更新白名单配置
3. 使用智能调优功能

### Q4: 系统支持哪些邮件格式？

- .eml格式（标准邮件格式）
- .msg格式（Outlook邮件）
- 直接粘贴邮件内容
