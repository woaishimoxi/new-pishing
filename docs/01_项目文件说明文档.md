### 3.3 API路由层 (`backend/app/api/`)

#### 3.3.1 `detection.py` - 检测分析API
- **作用**: 核心邮件检测接口
- **API端点**:
  - `GET /api/detection/health` - 健康检查
  - `POST /api/detection/analyze` - 邮件分析
  - `POST /api/detection/upload` - 上传邮件文件
- **功能流程**: 解析邮件 → 特征提取 → 模型检测 → 溯源分析 → 保存结果

#### 3.3.2 `alerts.py` - 告警管理API
- **作用**: 检测记录管理
- **API端点**:
  - `GET /api/alerts` - 获取告警列表（分页）
  - `GET /api/alerts/<id>` - 获取单条告警详情
  - `DELETE /api/alerts/<id>` - 删除单条告警
  - `DELETE /api/alerts/batch` - 批量删除告警

#### 3.3.3 `config.py` - 配置管理API
- **作用**: 系统配置管理
- **API端点**:
  - `GET /api/config` - 获取配置
  - `POST /api/config` - 更新配置
  - `GET /api/config/test` - 测试VirusTotal连接
  - `GET /api/config/test-email` - 测试邮箱连接

#### 3.3.4 `stats.py` - 统计数据API
- **作用**: 检测统计信息
- **API端点**:
  - `GET /api/stats/overview` - 获取概览统计
  - `GET /api/stats/daily` - 获取每日统计

#### 3.3.5 `email.py` - 邮件获取API
- **作用**: 邮件收取功能
- **API端点**:
  - `POST /api/email/fetch` - 从配置的邮箱获取邮件

#### 3.3.6 `system.py` - 系统管理API
- **作用**: 智能配置和性能监控
- **API端点**:
  - `POST /api/system/auto-tune` - 智能参数调优
  - `GET /api/system/performance` - 获取性能指标
  - `GET /api/system/performance/daily` - 获取每日性能
  - `GET /api/system/performance/report` - 生成性能报告
  - `POST /api/system/performance/export` - 导出性能报告

#### 3.3.7 `docs.py` - API文档
- **作用**: Swagger API文档生成
- **功能**: 自动生成API文档界面

### 3.4 业务逻辑层 (`backend/app/services/`)

#### 3.4.1 `detector.py` - 邮件检测引擎
- **作用**: 核心检测逻辑
- **主要功能**:
  - 加载LightGBM模型（39维特征）
  - 基于规则的检测（demo模式）
  - 多维特征分析
  - 风险等级判定（PHISHING/SUSPICIOUS/SAFE）
- **关键类**: `DetectionService`
- **特征维度**: 39维（邮件头8维 + URL14维 + 文本7维 + 附件5维 + HTML5维）

#### 3.4.2 `feature_extractor.py` - 特征提取器
- **作用**: 从邮件中提取检测特征
- **主要功能**:
  - 邮件头特征提取（SPF/DKIM/DMARC验证）
  - URL特征提取（域名年龄、HTTPS、短链接等）
  - 文本特征提取（紧急词汇、金融词汇等）
  - 附件特征提取（可执行文件、双重扩展名等）
  - HTML特征提取（隐藏链接、表单、iframe等）
- **关键类**: `FeatureExtractionService`

#### 3.4.3 `url_analyzer.py` - URL分析器
- **作用**: URL安全分析
- **主要功能**:
  - 域名白名单检查
  - VirusTotal API集成
  - WHOIS域名年龄查询
  - 可疑关键词检测
  - 品牌仿冒检测
- **关键类**: `URLAnalyzerService`

#### 3.4.4 `traceback.py` - 溯源分析器
- **作用**: 邮件来源追踪
- **主要功能**:
  - 邮件头Received链解析
  - 源IP提取和地理位置查询
  - 黑名单检查
  - URL重定向链追踪
  - 攻击链构建
  - IOC匹配
- **关键类**: `TracebackService`

#### 3.4.5 `email_parser.py` - 邮件解析器
- **作用**: 解析原始邮件格式
- **主要功能**:
  - 解析邮件头（发件人、收件人、主题等）
  - 提取邮件正文（纯文本和HTML）
  - 提取URL链接
  - 提取附件信息
- **关键类**: `EmailParserService`

#### 3.4.6 `email_fetcher.py` - 邮件获取器
- **作用**: 从邮件服务器获取邮件
- **主要功能**:
  - IMAP/POP3协议支持
  - 邮件服务器连接
  - 批量邮件获取
  - 未读邮件标记
- **关键类**: `EmailFetcherService`

#### 3.4.7 `auto_tuner.py` - 智能配置器
- **作用**: 自动参数调优
- **主要功能**:
  - 分析企业邮件模式
  - 推荐最优检测参数
  - 自动应用配置
- **关键类**: `AutoTuner`

#### 3.4.8 `performance_monitor.py` - 性能监控器
- **作用**: 系统性能监控
- **主要功能**:
  - 检测响应时间记录
  - 溯源耗时统计
  - 生成性能报告
  - 每日统计汇总
- **关键类**: `PerformanceMonitor`

#### 3.4.9 `sandbox_analyzer.py` - 沙箱分析器
- **作用**: 附件安全分析
- **主要功能**:
  - 可疑附件识别
  - VirusTotal沙箱查询
  - 恶意代码检测
